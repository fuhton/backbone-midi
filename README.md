## Backbone MIDI integration

* Enable the MIDI api in chrome here: chrome://flags/
* npm install
* gulp
* http://localhost:8080/
* All information will be in the console



###MIDI commands

* 0x80     Note Off
* 0x90     Note On
* 0xA0     Aftertouch
* 0xB0     Continuous controller
* 0xC0     Patch change
* 0xD0     Channel Pressure
* 0xE0     Pitch bend
* 0xF0     (non-musical commands)
